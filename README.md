# Surfaise
Solving PDEs on parametrized surfaces using FEniCS.

To install:
```
sudo python3 setup.py install
```
or
```
python3 setup.py install --user
```

Demonstrations to appear...

#### Dependencies
* FEniCS (Dolfin, UFL, ...)
* SymPy
* NumPy
* json
* cloudpickle
